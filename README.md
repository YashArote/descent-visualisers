# Descent Visualisers

Interactive labs designed to visualize the behaviour of various Machine Learning algorithms.

---

## Gradient Descent Visualiser

A fast, interactive tool to visualize how different gradient descent algorithms navigate complex loss surfaces in real time.

### Features
-  Explore optimizers: **Vanilla SGD**, **Momentum**, **Adagrad**, **RMSProp**, **Adam**
-  Real-time 3D visualizations using **Plotly**
-  Input custom loss surfaces using different functions
-  Adjust hyperparameters: learning rate, momentum, decay, beta values
-  Play, pause, reset animations and reinitialize points

## Reinforcement Learning Visualiser

Exploring how various Reinforcement Learning algorithms like QL, DQL, and PPO learn and act upon custom simulated environments.

### Environments & Features

**Maze Environment**
- Trained using tabular Q-Learning  
- Adjustable hyperparameters  
- Editable environment configuration  
- Visualised training steps  
- Play mode to observe trained agents  

**CartPole Environment**
- Trained using Deep Q-Learning (DQL) and Proximal Policy Optimization (PPO)  
- Adjustable hyperparameters  
- Environment parameter tuning  
- Visualised training steps  
- Play mode to observe trained agents
